# Учебный проект: preprocessing pipeline для ML

## Цель работы
Показать на небольшом Java-проекте, как связать структуры данных и простые алгоритмы в единую систему предварительной обработки данных перед задачами машинного обучения.

## Постановка задачи
Нужно построить pipeline, который:
1. Загружает объекты из CSV.
2. Индексирует признаки для быстрых диапазонных запросов.
3. Находит аномалии по Z-score.
4. Удаляет аномалии из набора.
5. Выполняет графовую кластеризацию очищенных данных.
6. Выводит итоговую статистику по этапам обработки.

## Используемые структуры данных
- `HashMap` — хранение точек по `id` и быстрый доступ к объектам.
- `TreeMap<Double, List<String>>` — индекс по признаку для range query.
- `List` — основной контейнер для объектов и кластеров.
- `Set` и `Deque` — обход графа похожести и поиск компонент связности.

## Этапы pipeline
1. **Загрузка CSV** (`CsvDataLoader`):
   - ожидается заголовок;
   - первый столбец — `id`;
   - остальные столбцы — числовые признаки.
2. **Обработка пропусков**:
   - выбрана стратегия **заполнения средним по признаку**;
   - если в строке есть нечисловое значение (и это не пустая ячейка), строка отбрасывается.
3. **Индексация признаков** (`FeatureStore`).
4. **Поиск аномалий** (`AnomalyDetector`) по выбранному признаку.
5. **Удаление аномалий** из данных.
6. **Нормализация для кластеризации** (`FeatureNormalizer`, Z-normalization).
   - нормализация применяется только перед кластеризацией.
7. **Графовая кластеризация** (`GraphClusterer`).
8. **Сводная статистика** (`PipelineResult`):
   - количество входных объектов;
   - количество аномалий;
   - размер очищенного набора;
   - число кластеров;
   - размеры кластеров.

## Формат CSV
Пример файла `data/sample_dataset.csv`:

```csv
id,age,income,spend
u1,22,35,30
u2,23,36,31
u3,24,37,
u4,45,80,75
u5,46,82,74
u6,90,300,5
u7,44,81,abc
```

- Пустая ячейка (как у `u3` в `spend`) заменяется средним по столбцу.
- Строка `u7` отбрасывается из-за нечислового значения `abc`.

## Как запустить
### Запуск через javac/java
```bash
javac -d out $(find src/main/java -name "*.java")
java -cp out ml.pipeline.Main data/sample_dataset.csv
```

=== Кластеры после очистки ===

=== Статистика ===

Найдено аномалий: 1

## Ограничения реализации
- CSV-парсер учебный: не разбирает экранированные запятые и сложные случаи формата.
- В кластеризации используется полный попарный расчёт расстояний (`O(n^2)`).
- Поиск аномалий сделан по одному выбранному признаку.
